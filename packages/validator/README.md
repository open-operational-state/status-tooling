# @open-operational-state/validator

Conformance validator for the Open Operational State standard. Validates operational-state responses against profiles, serializations, and conformance levels.

## Purpose

This package provides validation logic that checks whether a given response conforms to the standard. It works with fixtures and test taxonomy from [status-conformance](https://github.com/open-operational-state/status-conformance).
