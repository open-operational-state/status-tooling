# @open-operational-state/discovery

Discovery client for the Open Operational State standard. Locates operational-state resources using link-based, well-known path, and DNS bootstrap discovery mechanisms.

## Purpose

This package provides a client that discovers operational-state resources for a given target, determining what endpoints exist, what profiles they implement, and how to access them.
